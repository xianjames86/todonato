Frequently Asked Questions

What is Facebook AppState?

AppState contains your Facebook session cookies that allow the tool to authenticate and perform actions on your behalf.

How do I get my AppState?

You can extract it using browser developer tools or specialized browser extensions. Refer to the C3C FBState Guide for detailed instructions.

Is this tool safe to use?

While the tool is functional, always be cautious when sharing session cookies. We recommend using a separate test account.
